# 🏥 Hospital Appointment System

A web-based Hospital Appointment System developed using PHP, MySQL, HTML, CSS, and JavaScript.

## ✨ Features

- User Registration
- User Login
- Book Appointments
- View Appointments
- Admin Panel
- Logout

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
